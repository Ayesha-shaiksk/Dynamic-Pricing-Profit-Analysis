# 🚀 Dynamic Pricing & Profit Optimization System

## 📌 Overview

This project builds a complete pricing intelligence system using real-world e-commerce data.
It analyzes price sensitivity (elasticity), simulates pricing strategies, and evaluates profit impact to support data-driven business decisions.

---

## 🎯 Business Problem

E-commerce platforms often increase sales but fail to optimize profit due to:

* Poor pricing strategies
* Ignoring demand sensitivity
* High logistics (freight) costs

This project solves that problem using analytics.

---

## ⚙️ Solution Approach

* Data preprocessing and feature engineering
* Price elasticity modeling (log-log regression)
* Dynamic pricing simulation (+10% scenario)
* Profit impact evaluation
* Freight cost risk analysis
* Interactive dashboard for decision-making

---

## 📊 Key Features

* 📉 Price vs Demand (Elasticity Analysis)
* 📈 Category-wise Profit Optimization
* 🧠 Automated Pricing Decision (Increase / Stable / Decrease)
* 🚚 Freight Cost Impact Analysis
* 🌍 Regional Profit Insights

---

## 🛠 Tech Stack

* Python (Pandas, NumPy, Statsmodels)
* Tableau
* Google Colab

---

## 📈 Key Insights

* Low elasticity categories allow safe price increase
* High freight costs significantly reduce net profit
* Pricing decisions must consider both demand and logistics
* Profit improved by ~10–12% using optimized pricing
